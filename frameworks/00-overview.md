## How to use this with an AI assistant

**1. Weekly rhythm:** Monday morning — read one file, answer one question, derive one action.

**2. Before any new initiative:** Check it against the diagnostic files (`02-diagnostics/`). If your 1-1-1 isn't dialed in, fix that first.

**3. Before any decision:** Write a memo (WAFAM — see `07-communication-and-marketing/01-memo-culture-wafam.md`). Template in `09-templates-and-worksheets/01-memo-template.md`.

**4. Quarterly:** Soft Shop review (`04-business-architecture/02-soft-shop-strategy.md`) and bottleneck audit (`02-diagnostics/03-bottleneck-finder.md`).

**5. Annually:** Skill stacking inventory — what new combination did we build that no one else can replicate?

## The three sentences that change everything

1. **"What is the one thing — if I took it away — would not be valuable for you anymore?"** → reveals your real value prop.
2. **"Assume I've already accomplished this goal. Does everything else still work?"** → finds bottlenecks before they find you.
3. **"No memo, no meeting."** → forces clarity, eliminates theater.
