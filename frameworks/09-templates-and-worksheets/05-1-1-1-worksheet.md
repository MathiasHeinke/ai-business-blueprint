## Step 3 — Consolidation (20 min)

For every variant that *isn't* the best, three options:

| Action | When sensible |
|---|---|
| **Pause** | Has potential, but disperses attention now |
| **Cut** | Delivers little, costs much |
| **Keep** | Clearly delivers significant share, can't go |

## Step 4 — Optimization plan (10 min)

Now only work on the 1-1-1:

### Traffic optimization — next 30 days
- [ ] What would lift volume by 30%?
- [ ] What would raise quality by 1 point?
- [ ] Which one optimization to try first?

### Conversion optimization — next 30 days
- [ ] Where do you see the biggest drop-off?
- [ ] Could you hire a specialized consultant to watch 5–10 calls?
- [ ] Which one test to try first?

### Delivery optimization — next 30 days
- [ ] Where do customers leave? Why?
- [ ] What's the most founder-dependent piece? How to reduce it?
- [ ] Which one improvement to prioritize?

---

## Step 5 — Scaling path (ongoing)

Once the 1-1-1 performs stably:

```
Phase 1 (now): Dial 1-1-1 → up to $300k–$500k
Phase 2: Add 2nd traffic source → up to $1M
Phase 3: Add another source / 2nd conversion → up to $3M
Phase 4: Breaks → diagnose → fix
```

→ Don't skip. Don't accelerate. Complexity is exponential, not linear.

---

## Re-evaluation triggers

| Signal | Action |
|---|---|
| 1-1-1 stagnant for 3 months | Identify bottleneck, hire specialized consultant |
| Best source breaks | Immediately: set up 2nd source |
| Customer complaints about delivery | System build-out before more traffic investment |
