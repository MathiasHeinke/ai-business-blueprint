# 30-60-90-Day Plan — Structured Rollout

> Building on `01-quick-wins-7-days.md`. The order isn't random — plumbing before water.

---

## Days 1–30 — Foundation

### Goal
Clarity, diagnosis, customer insights, established memo culture.

### Track 1 — Customer Discovery
- [ ] 10 customer interviews (see `../03-customer-discovery/`)
- [ ] Pattern analysis, value-prop hypothesis
- [ ] Validate hypothesis with 3 more customers

### Track 2 — Diagnosis to Action
- [ ] Address identified bottleneck specifically
- [ ] Make cuts (at least 3 service lines / activities)
- [ ] Consolidate the 1-1-1 — keep only the performing variant

### Track 3 — Memo culture live
- [ ] At least 4 memos per week
- [ ] Every meeting WAFAM-compliant (or skipped)
- [ ] Memo archive navigable

### Track 4 — Financial architecture
- [ ] Define and implement lifestyle freeze
- [ ] Set up auto-savings transfer
- [ ] 6-month runway plan on the roadmap

### Success criteria (Day 30)
1. Clear value-prop hypothesis based on customer voices
2. Bottleneck addressed or in active solving mode
3. Memo culture embedded
4. Lifestyle freeze defined and started
5. 1–3 strategic cuts executed

---

## Days 31–60 — Build & Test

### Goal
Plumbing finished, hypotheses tested, A-player recruiting starts.

### Track 1 — Systems
- [ ] Delivery process SOPed (top-3 documented)
- [ ] New customer onboarding defined + tested
- [ ] Pre-flight checklist (`../02-diagnostics/04-plumbing-before-water.md`) ticked

### Track 2 — A-player Recruiting
- [ ] Top gap defined (pain or growth?)
- [ ] A-player JD (`../05-people-and-talent/02-a-player-job-descriptions.md`) created
- [ ] 5–10 profiles actively contacted
- [ ] First interviews

### Track 3 — Email list build-out
- [ ] Lead magnet defined + created
- [ ] Email tool selected + set up
- [ ] Welcome sequence (5–7 emails) written
- [ ] Capture flow on main site live

### Track 4 — Content rolls
- [ ] Weekly email cadence
- [ ] Weekly content on 1 channel
- [ ] First conversion data collected

### Success criteria (Day 60)
1. SOPs for top 3 processes
2. At least 1 active A-player hire pipeline
3. Email list live with first subscribers
4. Weekly send cadence established
5. First conversion data

---

## Days 61–90 — Scale & Validate

### Goal
Test growth, set up Soft Shop, build optionality.

### Track 1 — Test growth
- [ ] Add a 2nd traffic channel to the 1-1-1 — only if first is stable
- [ ] Actively optimize conversion (1 specialized consultant?)
- [ ] Transfer skill to a second person — if skills bottleneck

### Track 2 — Prep Soft Shop
- [ ] Research 5 plausible buyers (`../04-business-architecture/02-soft-shop-strategy.md`)
- [ ] Build package (P&L, one-pager, growth story)
- [ ] First outreach emails

### Track 3 — Allies & coaches
- [ ] 10-10 forever list finalized
- [ ] At least 1 strategic coach engaged (see `../05-people-and-talent/06-asking-for-help.md`)
- [ ] Per ally: concrete investment for next 90 days

### Track 4 — Low-phase preparation
- [ ] Low-phase plan documented (`../06-mindset-and-wealth/05-persistence-aquaman.md`)
- [ ] Allies know they serve as anchors
- [ ] Buffer at target level or with a clear path

### Success criteria (Day 90)
1. Growth on the 1-1-1 measurable (or clear bottleneck identified)
2. A-player hired or in final funnel
3. Email list with detectable engagement
4. Soft Shop initiated or planned
5. Coach / mentor in 10-10 relationship
6. Low-phase plan exists
