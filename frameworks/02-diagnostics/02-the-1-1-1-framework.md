# The 1-1-1 Framework

> *"One traffic source, one conversion mechanism, one delivery channel. The 1-1-1 will probably get you to a $300,000 business very quickly if you can dial it in."* — Sam Shvatsa

## The principle

Before you diversify, dominate **one** variant. Complexity multiplies. A clogged funnel across 5 channels isn't 5x harder to diagnose — it's 25x harder.

### The three "ones"

| 1 | Examples |
|---|---|
| **1 traffic source** | Paid Ads / Organic Content / Outreach / JV |
| **1 conversion mechanism** | 1:1 call / Self-serve funnel / Demo / Form |
| **1 delivery channel** | Facebook group / 1:1 / Online course / Done-for-you |

### Worked example

> *Leads via paid ads → conversion on a 1:1 call → delivery in a Facebook group.*

If the ads break, you're done. But you at least know the ads are working — and you can improve them.

## How to look at it

1-1-1 is **not the end state**. It's the diagnostic and scaling baseline. Once dialed: add a 2nd traffic source, then a 3rd. Until something breaks. That's when it becomes clear what to simplify or replace next.

## How to apply it

**Audit:**

| Area | How many active? | Which performs best? | What to cut? |
|---|---|---|---|
| Traffic sources | __ | __ | __ |
| Conversion methods | __ | __ | __ |
| Delivery channels | __ | __ | __ |

**Rule:** If you have >1 in any area without a clear top performer before the 1-1-1 is dialed → consolidate.

## Scaling path

```
Phase 1: Dial the 1-1-1 (target: $300k business)
   ↓
Phase 2: Add a 2nd traffic source (target: $1M)
   ↓
Phase 3: Add a 3rd source / 2nd conversion (if needed)
   ↓
Phase 4: It breaks → diagnose → fix
```

## Diagnostic when broken

If the system isn't performing, isolate:
1. Traffic? (volume too low)
2. Conversion? (script / demo / funnel)
3. Delivery? (output / retention / reviews)

A single specialized consultant per bottleneck can often lift the metric 5–10 percentage points — *without changing anything else*.

## Concrete actions

- [ ] List every current traffic source (all of them, honestly).
- [ ] Identify the best performer (if unclear → collect a month of data).
- [ ] Pause the others — don't delete, just halt.
- [ ] Same for conversion and delivery.
- [ ] 90 days exclusively dialing the 1-1-1.
- [ ] Only then add a second variant.
