# AI Prompts — Ready to Copy & Paste

> Tested against Claude, ChatGPT, Gemini, Cursor, and Continue. Replace `[brackets]` with your specifics.

The prompts below assume you've installed the **SOUL.md** character (paste it once into your system prompt or custom instructions). If you haven't — paste SOUL.md first, then use any of these.

## Quick-start (use this if nothing else)

```

My business: [2–3 sentences about what you do, who for, current revenue/size].
```

## 01 — Core Principles

### Find what you're doing because you *can* (not because of leverage)
```
Apply the Curse of Capability lens to my business.

Here are my current service lines / products / activities:
- [list 5–10]

For each, score: revenue share / effort share / scalability.
Then mark which ones I'm running because I CAN, not because they're
strategic. Recommend 3 cut candidates.
```

### Force yourself to simplify
```
Stress-test my business with the Simplicity-as-Strategy filter.

In one sentence: what's our value proposition? [paste here, or write
"I don't know"]

If "I don't know" — first run me through customer-discovery exercise.
If I have one — show me how every active initiative either supports
or violates it. List violations.
```

### Audit your week's attention
```
My current top 3 priorities this quarter are:
1. ___
2. ___
3. ___

Here's how I spent last week (rough breakdown):
[paste calendar summary or guess in %]

Where am I leaking attention? What gets cut this week so I can
double down on top 1?
```

## 02 — Diagnostics

### Find your bottleneck (TSS audit)
```
Run the Traffic-Systems-Skills diagnostic on my business.

I'll describe each, you score 1–10 and identify which is the bottleneck.

Traffic: [describe lead sources, volume, quality]
Systems: [describe what happens after a lead arrives, conversion process]
Skills: [describe who delivers, founder dependency, transferability]

Output: which is THE bottleneck this month, and what's the one fix
to focus on for the next 30 days.
```

### Audit your 1-1-1
```
Audit my 1-1-1 setup.

Traffic sources currently running: [list]
Conversion methods currently used: [list]
Delivery channels currently used: [list]

Identify the best performer in each. Recommend which underperformers
to pause / cut / keep. End with a 90-day consolidation plan.
```

### Find bottlenecks before they find you
```
Run the "Assume Success" exercise.

Assume one of these scenarios is true next month:
- My revenue doubles
- My customer count triples
- I'm out of office for 4 weeks

For each: what breaks first? Then prioritize what to fix BEFORE
that growth would arrive.
```

## 03 — Customer Discovery

### Generate your interview script
```
I want to interview 10 of my existing customers using the two magic
questions from the AI Business Blueprint.

My business: [short description]
Common customer profile: [who they are]

Generate:
1. An opening script (warm, low-pressure, gets permission)
2. The two questions phrased for my context
3. Three follow-up prompts for vague answers
4. A closing line

Output as a script I can read on a call.
```

### Pattern-analyze your interview results
```
Here are my customer interview answers (10 customers, 2 questions each):

[paste raw notes]

Find patterns. Specifically:
- Which take-away answer recurs most often?
- Which word or concept appears in >50% of answers?
- Which add-on wishes show a pattern?

End with a one-sentence value-prop hypothesis based on what you found.
```

### Validate your value prop in one sentence
```
My current value prop draft: "[draft]"

Stress-test it with these questions:
1. Is it concrete and measurable?
2. Does it scale emotionally (i.e. could be reframed as "what would
   you do with...")?
3. Could every initiative be filtered through it (yes/no)?

If it fails any of these, rewrite it. Then give me 3 alternative phrasings.
```

## 04 — Business Architecture

### Build the Soft Shop package
```
Help me prepare a Soft Shop package for plausible buyers.

About my business:
- Industry: ___
- Revenue: ___
- Margin / EBITDA: ___
- Team size: ___
- Top 3 differentiators: ___
- Concentration risk: ___

Generate:
1. A one-pager (text only, structured)
2. List of 5 plausible buyer types (strategic, financial, complementary)
3. Outreach email template
4. The "magic question" script for the call
5. Documentation template for after the call
```

### Sellability audit
```
Run a Sellability Audit on my business.

Score (1–10) and explain gaps for:
- Documented processes
- Customer concentration risk (>20% from one client?)
- Delivery skill on more than one person
- Recurring revenue / predictability
- Brand IP, domains, contracts clean
- Financials clean

End with: the two lowest scores become my next-quarter priorities.
What concrete actions would lift each by 2 points?
```

## 05 — People & Talent

### Build an A-player JD
```
I need to hire [role/area].

Here's my raw, unfiltered pain (or aspiration):
[free-write 5–10 minutes worth of raw text — what's broken, what
would be magical if solved]

Translate this into an A-player JD using the structure in
frameworks/05-people-and-talent/02-a-player-job-descriptions.md.

Include:
- Real outcomes (not tasks)
- Honest "you don't fit if..." section
- A specific self-selection question
- Compensation range placeholder I'll fill in
```

### 10-10 Forever inventory
```
Help me run a 10-10 Forever inventory.

For the next 10 minutes I'll list 10 people and answer for each:
- Why I'd bet on them for 10 years
- What I'd invest in them
- What they bring back

Ask me each set of questions one person at a time. After all 10:
- Who's missing from the list? (Skills / domain / energy gaps)
- Who's a candidate I should add?
- Where am I currently under-investing?
```

## 06 — Mindset & Wealth

```
Help me set up a lifestyle freeze.

My current monthly fixed costs:
- Housing: ___

- Subscriptions: ___
- Other: ___

```

### Skill stacking diagnosis
```

[free-write 15–20 skills, even unrelated]

Identify:
1. Top 5 strongest
2. Top 3 unusual (rare relative to my industry)
3. Three rare combinations I could position around
4. For each combination: does a market exist, and where would I find it?
```

## 07 — Communication & Memos

### Generate a memo from raw thoughts
```
Convert my raw thoughts into a memo using the standard template.

[paste unstructured text]

Structure: TL;DR, Story so far, The Issue, Options (with pro/con),
Recommendation, Risks, Open questions, Next steps.

Mark gaps explicitly where I omitted information.
```

### Decision memo (fast)
```
I need to decide: [one-sentence decision].

Background: [3 bullets]

Walk me through Branson's 4 steps:
1. Context (what we know, what we don't)
2. Issue (the actual problem)
3. Risk (worst case, can we live with it)
4. Map (next steps, who/what/when)

Then write the decision memo for my team.
```

## Tips for getting better answers

1. **Start with SOUL.md.** It changes the AI's tone permanently.
2. **Reference framework files by path.** Helps the AI ground its advice.
3. **Use real numbers.** "We do $X" beats "We do well."
4. **Ask for outputs, not opinions.** "Generate a 30-60-90 plan" beats "What should I do?"
5. **Push back if the AI gets soft.** SOUL.md should prevent it, but if you see "great question" or "I'd be happy to" — remind it: *"Stay in character."*
